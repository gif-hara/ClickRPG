# ゲームサイクル

- 敵をクリックするとダメージを与える
- HPが0になったら経験値を取得する

# ステータス

- 攻撃力
- クールタイム
- 会心率
- 装備可能スロット

# スキル

- 攻撃力が{Lv * 10}上昇する
- クールタイムが{Lv * 1}%減少する
- 会心率が{Lv * 0.5}%上昇する
- 獲得する経験値が{Lv * 0.5}%上昇する
- 攻撃を3回する度に攻撃力 + {Lv * 0.1}の威力を持つ隕石が降ってくる
- バトル中、自動的にクリックする
- 会心ダメージが{Lv * 0.05}%上昇する
- 会心が発生した時にクールタイムが20 + {Lv * 0.1}%減少する
- 低確率で攻撃力の{Lv * 0.5}%のダメージを与える
- 会心率が100%を越えている場合、会心ダメージが{Lv * 1}%上昇する
- プレイ時間（秒） * {Lv * 0.5}%の数値を攻撃力に加算する
- 倒した敵の数 * {Lv * 0.5}%の数値を攻撃力に加算する
- バトル開始時にマウスポインタが生成される。このマウスポインタの攻撃力は素の(攻撃力 * {Lv * 0.1}) / 2である
