# ゲームサイクル

- 敵をクリックするとダメージを与える
- HPが0になったら経験値を取得する

# ステータス

- HP
- 攻撃力
- クールタイム
- 会心率
- 装備可能スロット
- 貫通力
- 弾速

# スキル

- ノーマルバレット
- ショットガン
- マシンガン
- スナイパー
- レーザー
- ホーミング
- HPが1上昇する
- 攻撃力が5上昇する
- クールタイムが10%減少する
- 会心率が10%上昇する
- 会心ダメージが5%上昇する
- 会心が発生した時にクールタイムが20%減少する
- 敵を5体倒す毎に攻撃力が2上昇する
- 敵を5体倒す毎に会心率が4%上昇する
- 弾の貫通力が1上昇する
- 弾速が10%上昇する
- 獲得する経験値が10%上昇する
- 敵を20体倒す毎にHPが1回復する
